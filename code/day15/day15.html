2、特殊字符
正则中的特殊字符
^ : 表示正则表达式的开始
$ : 表示正则表达式的结束
. : 表示任意一个字符
+ : 表示其前面紧挨着的字符至少出现一次，至多无数  等价{1,}
? : 表示其前面紧挨着的字符至少出现0次，至多出现一次  等价{0,1}
* : 表示其前面紧挨着的字符至少出现0次，至多无数   等价{0,}

\w : 表示字母、数字、下划线中的任意一个字符   （任意一个单词字符）
\W:  表示除了 字母、数字、下划线中的任意一个字符 （非单词字符）
\d : 表示 0--9 之间的任意一个数字   [0-9]
\D : 表示 除了0--9 之间的任意一个字符（非数字）
\s : 表示空白字符（空格、tab键）
\S : 表示非空白字符

{m,n} : m<n   表示括号前面紧挨着的字符至少出现m次，至多出现 n 次  （m、n均是正整数）
{m,}  :     表示括号前面紧挨着的字符至少出现m次，至多无数
{m}   :     表示括号前面紧挨着的字符只能出现m次
[  ]    : 表示括号内的任意一个字符
[^ ]    : 表示非括号内的任意一个字符
[eu21] : 表示 e 或  u  或 2  或 1
[1-9]  :  - 表示一个范围 ，表示1--9之间的任意一个数字
[^0-9]  等价  \D  表示非数字
()  ：  表示 优先级
  |   :  表示  或者
  \  :  表示转义 （如果正则表达式中有些特殊字符想表达普通字符的含义，这些特殊字符需要转义）   \. ---> 表示一个 .  
  表示一个中文 ： [\u4e00-\u9fa5]        [\u2e80-\u9fff]---unicode编码

3、正则中的相关方法

正则表达式相关方法
test() 检验一个字符串是否符合某一个正则规范，如果符合返回true，否则返回false
用法： reg.test(str)
----------------------------------------
exec()  根据正则表达式查找，结果会返回一个长度为1的数组 （数组只有一个值）
用法： reg.exec(str);
 

正则中字符串的方法：
-----------------------------------------
search方法     返回与正则表达式查找内容匹配的第一个子字符串的位置。
 用法：  str.search(reg);
 
---------------------------------------------------------------

match() 方法   使用正则表达式模式对字符串执行查找，并将包含查找的结果作为数组返回。
 用法：  str.match(reg) 
         

-------------------------------------------------------

str.replace(原字符串,新字符串)

replace（）   通过正则表达式替换，返回结果 是原字符串被替换后的新字符串  

用法：  str.replace(正则表达式,要替换的新字符串)      如果想实现全局替换，在正则表达式上加一个修饰符 g
