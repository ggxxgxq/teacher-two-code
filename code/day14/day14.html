2、cookie的生存期
生存期 ：cookie的文件信息保存时间
设置 ： 
document.cookie = "键=值;expires=过期时间"  过期时间格式 ：标准时间格式  
设置3天后的时间 ： 
var d = new Date();
d.setDate( d.getDate()+3 );

cookie生存期总结：
设置生存期后，cookie文件默认在过期时间后消失
如果没有设置生存期，cookie文件在浏览器关闭后就消失（会话结束后）
如果将系统时间改为过期时间之后，属于假删除  
如果要彻底删除cookie ：  将生存期设置为-1   或者 将cookie键对应的值 改为 ""

4、cookie总结
cookie参数 ： 
键   必须
值   必须
生存期  expires
域名 domain  :  127.0.0.1
路径  path  :   /...
document.cookie = "uname=lichune;expires=d;domain=127.0.0.1;path=/"

cookie 存储数据量小 最大不能超过 4kb   cookie不能超过50个    安全性低
cookie 不能跨域存储  （跨文件夹不可以）

5、正则 ： 正确的规则

正则 用来规范表单数据的填写是否合法
正则表达式组成 ： 普通字符 和  特殊字符组成的 
特殊元字符(单字符)：  $   +   .   ^  
特殊组合字符 ： \d   \w    \s    ....
  
定义 ： 
var reg = /普通字符或特殊字符/

5、正则中的特殊字符
^ : 表示正则表达式的开始
$ : 表示正则表达式的结束
. : 表示任意一个字符
+ : 表示其前面紧挨着的字符至少出现一次，至多无数  等价{1,}
? : 表示其前面紧挨着的字符至少出现0次，至多出现一次  等价{0,1}
* : 表示其前面紧挨着的字符至少出现0次，至多无数   等价{0,}

\w : 表示字母、数字、下划线中的任意一个字符   （任意一个单词字符）
\W:  表示除了 字母、数字、下划线中的任意一个字符 （非单词字符）
\d : 表示 0--9 之间的任意一个数字   [0-9]
\D : 表示 除了0--9 之间的任意一个字符（非数字）
\s : 表示空白字符（空格、tab键）
\S : 表示非空白字符

{m,n} : m<n   表示括号前面紧挨着的字符至少出现m次，至多出现 n 次  （m、n均是正整数）
{m,}  :     表示括号前面紧挨着的字符至少出现m次，至多无数
{m}   :     表示括号前面紧挨着的字符只能出现m次
[  ]    : 表示括号内的任意一个字符
[^ ]    : 表示非括号内的任意一个字符
[eu21] : 表示 e 或  u  或 2  或 1
[1-9]  :  - 表示一个范围 ，表示1--9之间的任意一个数字
[^0-9]  等价  \D  表示非数字
()  ：  表示 优先级
  |   :  表示  或者
  \  :  表示转义 （如果正则表达式中有些特殊字符想表达普通字符的含义，这些特殊字符需要转义）   \. ---> 表示一个 .  
  表示一个中文 ： [\u4e00-\u9fa5]        [\u2e80-\u9fff]---unicode编码
