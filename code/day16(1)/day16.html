2、块级作用域  let  （重点）
   1--let只能作用在{}内部  
   2--let定义的变量不能重复声明  
   3--let不能实现变量提升
   4--let声明的变量  变量未声明之前该变量不可以使用  这种现象 叫做 "暂时性死区"
3、for of 和  for  in  区别：
   for  of  是遍历值     不能遍历 json
   for  in  是遍历键
4、字符串模板   `    `   即可以识别变量  也可以识别函数的调用    用来做字符串拼接 （重点） 
5、常量定义关键字const
一般常量名用大写字母 和变量区分

2、箭头函数   =>  

定义 ：
（）　=>　｛　函数体　｝

　
带有一个参数　：　　小括号可以省略　
a => { alert(a) }   或  (a) => { alert(a); }

带两个或以上参数格式：
（a,b） => { return a + b ;}


3、解构赋值

1、可以一次性定义多个变量
2、可以让一个函数返回多个值  return   [  ]
3、可以实现两个变量的交换（不借助第三个变量）
4、可以作用在函数的参数上，参数传值可以不按照书写顺序传递

4、Array.from    将伪数组转成数组

用法  ：   Array.from( 伪数组 );  返回一个转成后的新数组  （只要有length属性  就可以将其转成数组）
伪数组 ： 有长度属性  取值也通过下标取值，不能直接使用数组的操作方法  push pop等


Array.from( 集合 , map函数 )
（扩展）第二个参数：实现map函数功能   将集合转成数组后，在对数组进行map操作，返回map操作后的新数组

5、Set和Map

Set集合特点 ：　自动去重    不会实现自动类型转换 
数组去重　：　
     Array.from( new Set([34,23,34,56,23,56,56,90]) );　　　　或　　[...new Set([34,23,34,56])]

set 集合方法 ：
add(num)   向集合添加数
delete(num)  删除集合中的某个数
has(num)  判断集合中是否含有某个数
clear()  清空集合   没有参数

    集合遍历
          set.forEach( (value) => { console.log( value ); } )



Map 集合 ：
定义 ：　var  map = new Map();    值以  键值对的格式存储
map集合方法：
     set()    为map集合设置值   map.set("键","值");
     get()    根据键获取值    map.get("键");
     delete（） 删除          map.delete("键")
     has()     判断是否含有某个键    map.has("键")
     clear()  清空集合       map.clear()
     map集合遍历：  for  of
     for( var value of map ){
        console.log( value[1] );
        //console.log( map.get(value[0]) );
     }
获取集合长度属性 ： size
